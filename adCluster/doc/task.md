# 연구노트 시스템 전환 작업 목록 (Task Checklist)

이 문서는 `plan.md`를 기반으로 현재 프로젝트 관리 시스템을 연구노트 시스템으로 전환하기 위한 구체적인 작업 목록을 제공합니다.

## 프로젝트 목표

현재 프로젝트 관리 도구로 보이는 시스템을 연구노트 시스템으로 전환하여, 연구자들이 논문 작성, 참고문헌 관리, 연구 데이터 정리 등을 효율적으로 수행할 수 있도록 개선
